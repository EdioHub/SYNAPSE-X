# SYNAPSE-X
Implementação do modelo SYNAPSE-X, desenvolvido como Trabalho de Conclusão de Curso em Business Intelligence e Analytics (MBA USP). Trata-se de uma arquitetura híbrida que combina autoencoders, Self-Organizing Maps (SOM) e K-Nearest Neighbors (KNN) para identificação de padrões em séries temporais financeiras. O modelo utiliza janelas deslizantes e variáveis exógenas para capturar similaridades entre diferentes estados de mercado, permitindo a geração de previsões direcionais e análise de comportamento histórico.
